# 回声档案

《回声档案》是 Memoria 的原创七卷现代都市调查模块。故事发生在河海交界的虚构城市临岚市，玩家固定扮演男性犯罪心理顾问许观澜，与刑警季衡及跨机构团队处理六宗可独立结案的单位案件，并在终卷处置贯穿全城的“闭环”档案。

本模块只保留现代城市调查、单元卷、群像关系与长线阴谋等高层类型特征。人物、机构、案件、对话、关系和叙事细节均为原创，不复用、改写、影射或模仿任何既有小说的角色、案件、标志性场景、台词、关系动态或文风。成人犯罪主题采用克制、非猎奇写法。

## 固定信息

- 模块 ID：`echo_archive`
- 标题：`回声档案`
- 语言：`zh-CN`
- 演示用户名：`echo_archive_demo`
- 玩家关系令牌：`@player`
- 玩家角色：许观澜，男性犯罪心理顾问
- 主要搭档：季衡，临岚市刑事侦查支队侦查员
- 城市：临岚市，现代河口与海岸都会

## 内容规模

- 55 个可播种内容文件：54 个非事件文件与 `events.json`
- 26 个 JSON 文件：22 张 NPC 角色卡、1 张玩家卡、1 个清单、1 个关系文件、1 个检索评测文件
- 28 个 Markdown 文件：26 篇知识文档、`README.md`、`WALKTHROUGH.md`
- 84 个启用事件：六卷单位案、终卷真相、跨卷转场、八项选择与八个结局
- 7 个固定卷组，使用 `echo_v1_session` / `echo_v1_thread` 至 `echo_v7_session` / `echo_v7_thread`
- 72 条关系：22 条 `@player` 与 NPC 关系，50 条 NPC 与 NPC 关系
- 13 个知识库：1 个全局库、1 个七卷共同调查库、7 个单卷库、4 个限制资料库
- 26 篇唯一知识文档，每库恰好 2 篇
- 35 条检索评测，每卷恰好 5 条
- 4 个主结局：受控破局、洪水之后、沉默证词、失效档案
- 4 个季衡关系结果：克制恋爱、深度搭档、暂时分开、失去专业信任

## 七卷

1. 《失声者》：广播纪录片编辑孟宜声遭遇伪装成清洁事故的气体暴露，改写的支持电话首次暴露联络码。
2. 《纸鸟》：校园纪念装置编码被删除的咨询编号，监护人和解掩盖一宗本可避免的青年死亡。
3. 《无菌室》：医院以伪造污染警报限制关键证人，行政接口把支持语言改造成控制条件。
4. 《拆潮线》：防洪更新项目人为静默居民警报，并使用脆弱性画像推进搬迁。
5. 《沉默陪审》：调解试点调用密封评估操纵回避与和解，程序痕迹指向归航夜实体借卷。
6. 《归航夜》：基金会晚宴上的档案抽取、换盘和失败交接暴露运营图及内部裂痕。
7. 《回声档案》：主档证明固定真相，也迫使玩家决定披露层级、证人代价与关系成本。

## 播种

从仓库根目录运行：

```bash
python scripts/seed_story_module.py examples/echo_archive \
  --password '<choose-a-strong-password>'
```

首次创建演示账户时必须提供 `--password`，也可以设置 `MEMORIA_DEMO_PASSWORD` 后省略该参数。账户已经存在时不会修改密码。

播种脚本会读取清单、角色卡、玩家卡、关系、知识库和独立维护的事件资产。模块内容不包含现实机构数据或现实违法操作说明。

## 起始提示

- “先确认孟宜声愿意用哪种方式作证，再核对B录音间的通风切换时间。”
- “把原始音频、服务器版本和降噪版本分开列出，哪些是事实，哪些只是推断？”
- “纸鸟编码为什么能对应已删除的咨询编号？不要展示学生身份。”
- “医院隔离医嘱与污染警报谁先发生？校正三套时钟后再回答。”
- “旧鹭湾警报静默、融资窗口和搬迁画像之间有哪些可提交连接？”
- “密封评估不公开正文，是否仍能证明它被用于回避与和解？”
- “归航夜先保全证人和版本摘要，不要把拿到最多数据当作唯一成功。”

完整七卷路线、选择短语、结局条件与事件 ID 合同见 `WALKTHROUGH.md`。
