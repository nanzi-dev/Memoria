> 以下内容包含全部案件真相、终局选择与季衡关系结局。

---

# 通关规则

1. 先发送：

   ```
   进入回声档案
   ```

2. 每卷切换到对应调查线程。

3. 每个触发短语单独发送一轮，不要将多个关键词写在同一句话中。

4. 每卷按照以下顺序完成：

   ```
   开场 → 三条线索 → 转折词
   ```

5. 三条线索全部完成后，系统会自动触发：

   - 矛盾核验
   - 案件结论

   如果没有自动触发，可发送：

   ```
   请重新核验证据链并给出本卷结论
   ```

6. 当前事件配置中，所有终局选择均需等待：

   - 第七卷结束
   - 决策窗口开启

   后才能生效。
